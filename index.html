Ichimoku Kinko Hyo（一目均衡表）指标详解

一、计算方法

线条名称计算公式默认周期
Tenkan-sen转换线(N日内最高价 + N日内最低价) / 29日
Kijun-sen基准线(N日内最高价 + N日内最低价) / 226日
Senkou Span A先行线A(转换线 + 基准线) / 2，向前位移26日26日位移
Senkou Span B先行线B(N日内最高价 + N日内最低价) / 2，向前位移26日52日+26日位移
Chikou Span滞后线收盘价，向后位移26日-26日位移

二、Kumo云带（Senkou Span A与B之间的区域）

绿色云（Bullish）：Span A > Span B，代表多头力量主导，云带构成支撑区。

红色云（Bearish）：Span A < Span B，代表空头力量主导，云带构成阻力区。

云带翻转：Span A与Span B交叉时，趋势可能发生转折。云带越厚（A与B距离越大），支撑/阻力越强。

三、五大核心信号

信号条件含义
价格 vs 云带价格在云上方 / 下方 / 云中多头趋势 / 空头趋势 / 震荡
TK金叉转换线上穿基准线短期动能转多，买入信号
TK死叉转换线下穿基准线短期动能转空，卖出信号
云带翻转Span A上穿/下穿Span B中期趋势转折确认
Chikou确认滞后线在价格上方/下方趋势得到滞后确认

四、三重确认交易系统

强势多头 = 价格 > 云带 AND Tenkan > Kijun AND 云带为绿色
强势空头 = 价格 < 云带 AND Tenkan < Kijun AND 云带为红色
中性/震荡 = 上述条件不满足时

五、与其他指标的配合逻辑

指标角色与Ichimoku配合方式
RSI超买超卖Ichimoku给出趋势方向，RSI寻找入场时机（回调至30-40买入）
MACD动能确认Ichimoku金叉 + MACD柱状图扩张 = 多头动能共振
布林带波动通道布林带触及下轨 + Ichimoku云带支撑 = 高概率反弹
ATR波动率ATR升高 + Ichimoku云带翻转 = 趋势加速确认
